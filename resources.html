MHE NMDS Data Entry Tool

The [MHE NMDS data entry tool] is a Microsoft Access Application which assists Jurisdictions to properly format and submit an MHE Data File into the Online Validator. A release of this tool is made available for each submission period (17/18, 18/19, 19/20 etc). The data entry tool is available at https://validator.com.au/mhe-nmds-data-entry-tool/index.html

MDS specifications documentation

Specifications for each of the data sets accepted by the Online Data Validator are available at https://docs.validator.com.au/
